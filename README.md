# Gemini & 監督：タグ実験場

> ＊＊Geminiさん：なんでも頼めば、すぐやってくれる、私の開発パートナー! 尊敬込めて、Geminiさんとお呼びしながら、きっちり寄りかかり、日々に開発中です。**

### 【開発の経緯】

アプリ開発中に、幾度となくHTMLの改変を迫られる。
ブラウザのF１２（デベロッパーツール）は優れもので、開発ド素人でも、毎日触っていると、幾分使い方が分かってくる。そうすると、そこでレイアウトやデザインを変えたりも出来るのだが、やはり、ソース全体を見ながらやるには、別個にタグテストできるソフトが欲しいなとなった。

そこで、Geminiさんに、「タグテストアプリ作ってや」と頼んだ。即、数秒で出来た！　数秒やで！　びっくりやね。
で、使いながら、ああして、こうしてと頼みながら、更に改変。まだまだ改変してもらいたいところはあるが、本業のアプリ開発が忙しく、この状態で止まってる。

でも、時々、開発アプリのコードを丸投げして、F12で開いたページ情報から「id」「class」「該当タグ名」などを、タグテストアプリで「Ctrl+F」検索使い拾いながら、デザインいじくったり、レイアウト変えてみたりして、遊んでる。

ソースコード利用し、自分専用にローカルで使えるタグテストアプリ作成すれば、オフラインで使えるからして、セキュリティ面で安心できるかも。
